# Smart-Expense-Analyzer-with-Alerts
Smart Expense Analyzer with Alerts

 Intermediate Task (Upgraded)

Problem Statement:

Write a Python program that:

1. Asks the user how many expenses they want to enter.

2. Uses a loop to take:

o Expense amount

o Expense category (Food, Travel, Shopping, Other)

3. Stores:

o All expense amounts in a list

o Category-wise totals in a dictionary

4. During input, check:

o If any single expense is greater than ₹500, print

“High expense detected”

5. After all inputs:

o Calculate and print total expense

o Calculate and print average expense

o Print category-wise totals

o Find and print the highest spending category

6. Use if / elif / else to print a spending status:

o Average ≤ ₹200 → Controlled Spending

o Average ≤ ₹400 → Moderate Spending

o Average > ₹400 → High Spending
